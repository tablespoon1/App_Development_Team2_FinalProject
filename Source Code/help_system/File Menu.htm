Below is the listing of FILE menu items used for operating the application:

Open Edge File: Opens a preceding edge file that can be loaded into the converter.

Open Save File: Opens an existing edge file that can be loaded into the converter. All modifications made earlier will also be restored.

Save: Saves current state of the file loaded into the application by default extension.

Save As: Saves current state of file loaded into the application by specific extension determined by the user

Exit: Closes the application; any changes made to an open files will be discarded.
