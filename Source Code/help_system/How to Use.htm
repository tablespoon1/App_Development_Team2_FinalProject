Open and edge file to be converted by using the 'File' menu system. Select either a new file or an existing edge save file.

When opened, the tables and field listing windows will be populated with data sourced from the edge file selected. These listings can be manipulated utilizing the interface up/down arrow buttons and radio buttons. Each one respectively allowing to change the order of each attribute and data type. Likewise, the field boxes and check boxes grant permissions to constraint the attirbutes and set characteristics.

Next, user's can define relationships between table sets. Such relationships that can be automically constructed are one to one, one to many or many to one, many to many, and self referencing table relationships.

Lastly, the user can convert the the existing edge file to another reference source such as XML output or another other apprpriate file definition. These definitions are extended class files of the program. Therefore, they need to be constructed or downloaded for use.
